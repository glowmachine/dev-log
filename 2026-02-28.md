# 28-02-2026, Learning HTML
For the past week, I've been learning HTML from Dave Gray. I think he did an amazing job teaching in a way that made me consider things that weren't obvious to me, like the importance of using semantic tags and ARIA.

For my first project, I decided to organize my git notes into a pure HTML page (tried to, there's a small amount of styling for legibility.)

The hardest part was going through and wrapping text in `<code>` and `dt`+`dd` tags. At first, I tried using `details`+`<summary>` blocks, but I thought there were so many commands to cover that needing to click on each one to reveal info would have made it less user-friendly.
